一、 课程目标


《雅思英语》课程是外国语学院为会计学院会计学（ACCA）专业所定制的一门通识公共必修


课程。此课程涵盖雅思听力、雅思口语、雅思阅读和雅思写作，对考试内容、提问方式、解题方法


和考试技巧进行逐一的讲解和全面的介绍，并且提供模拟试题，以帮助学习对雅思考试的形式、要


求、难度等有一个全面的了解，提高使用英语的能力。在学习雅思英语的过程中，通过使用富含中


西方文化的听力、阅读等材料增加学生在社会、文化、科学等领域的知识储备，拓宽国际视野，提


升综合文化素养，树立正确的世界观、人生观、价值观。并将中国特色社会主义思想融入课堂，旨


在增强学生社会主义核心价值观认知，提升学生的文化自信，激发学生使命感和责任心。


通过理论教学和实践活动，达到以下课程目标：


课程目标 1：发展学生国际视野和人文关怀。


课程目标 2：提高学生语篇阅读和听力理解能力。


课程目标 3：提高学生说和写的语言应用能力。


课程目标 4：提高学生举一反三的思维能力和合理安排自主练习的能力。


课程目标 5：坚定学生的理想信念，以爱党、爱国、爱社会主义、爱人民、爱集体为重点。 课

本课程详细教学内容和方法阐述如下：


**1.** 测试方向与评分要求简介


（ **1** ）主要内容


    - 雅思考试简介


    - 雅思考试四部分基本简介


    - 雅思考试四部分评分标准介绍


    - 雅思考试四部分范例介绍


（ **2** ）教学重点： ①雅思考试四部分介绍；②雅思考试四部分评分标准介绍。


（ **3** ）教学难点： ①雅思考试四部分特别的评分标准；②雅思考试中心出题思路。


（ **4** ）教学要求： 能够通过雅思考试整体以及其四部分的介绍，了解雅思考试的基本结构和考


试思路，能够清楚雅思考试过程中将会遇到何种挑战，能够明白在每一个考试部分需要完成何种题


型。


思政融合点 **1** ：通过雅思考试的基本介绍以及各部分评分标准的介绍，与学生常见的英语能力


测试（中高考、四六级、考研等）进行比较，从而了解潜在的中西方文化差异。在文化比较中，加


强文化认同，加强文化自信，实现立德树人的根本任务。


**2.** 听力题型与技巧


（ **1** ）主要内容


    - 听力考试详细介绍


    - 听力考试各种题型介绍


    - 听力考试基本技巧


    - 听力考试真题讲解


（ **2** ）教学重点： ①听力考试各种题型介绍；②听力部分基本考试技巧讲解。


（ **3** ）教学难点： 听力部分基本考试技巧。


（ **4** ）教学要求： 通过课堂讲授、线上视频学习、真题讲解，学生能够熟练地将雅思考试的听


力题目进行分类，能够了解不同题型相应的解题技巧，能够知道听力考试不同部分的常见的材料内


容，能够适应雅思听力考试的语音、语调、以及语速。


思政融合点 **2** ：在听力部分的讲解中，通过明确课程教学目标和育人使命，挖掘课程教学中蕴


含的思政元素，实现英语听力技巧与文化素养的协同培养。打破传统单一的“技巧教学观”，通过将


中西文化对比、英语文化介绍和语言习惯差异等内容渗透于课程教学中，使学生在参与英语听力课


堂学习过程中，形成良好的人文思辨精神和正确的文化观。以“隐性育人”方式为基础，对听力文本


内容进行拓展、延伸，实现语言能力培养与价值引导的深度融合。


**3.** 阅读题型与技巧


（ **1** ）主要内容


    - 阅读考试详细介绍


368


    - 阅读考试各种题型介绍


    - 阅读考试基本技巧


    - 阅读考试真题讲解


（ **2** ）教学重点： ①阅读考试各种题型介绍；②阅读部分基本考试技巧讲解。


（ **3** ）教学难点： 阅读部分基本考试技巧。


（ **4** ）教学要求： 通过课堂讲授、线上视频学习、真题讲解，学生能够熟练地将雅思考试的阅


读题目进行分类，能够了解不同题型相应的解题技巧，能够知道阅读考试不同部分的常见的材料内


容，能够适应雅思阅读考试的篇幅、体裁、以及单词难易程度。


思政融合点 **3** ：在阅读部分的讲解中，通过使用中国题材的补充阅读材料，增强学生的跨文化


交际意识和交际能力，满足国家、社会、学校和个人发展的需要。坚持“社会主义核心价值观”，通


过价值塑造，传授知识，培养能力，使学生成为具有中国情怀的国际化人才。


**4.** 写作题型与技巧


（ **1** ）主要内容


    - 写作考试详细介绍


    - 写作考试两种题型介绍


    - 写作考试基本技巧


    - 写作考试真题讲解


（ **2** ）教学重点： ①写作考试两种题型介绍；②基本考试技巧讲解。


（ **3** ）教学难点： 写作部分基本考试技巧。


（ **4** ）教学要求： 通过课堂讲授、线上视频学习、真题讲解，学生能够知晓雅思写作的两个部


分的基本形式，能够了解两个部分的基本解题技巧，能够大体了解雅思大、小作文两部分的常见话


题分类。能够适应学术论文写作在形式、结构、字数限制等方面的特点。


思政融合点 **4** ：在写作部分的讲解中，不仅能够有效帮助学生打好语言基本功，还引导学生探


析事物本质，从而培养学生思辨能力，陶冶道德情操、提高人文素养、促进自我完善。讲授英语学


术论文写作技巧时，从句子到篇幅，对照意义相同但写作方式不同的英文表达，学生会意识到语言


背后的意识形态与文化差异，从而通过比较和思考增强对语言的理解。从写作方式和写作技巧的差


异中，学生能辩证地认识中英两种文化差异， 激发学生的政治认同和家国情怀。


**5.** 口语题型与技巧


（ **1** ）主要内容


    - 口语考试详细介绍


    - 口语考试各部分介绍


    - 口语考试基本技巧


    - 口语考试真题讲解


（ **2** ）教学重点： ①口语考试各部分题型介绍；②口语考试基本技巧讲解。


（ **3** ）教学难点： 口语部分基本考试技巧。


369


（ **4** ）教学要求： 通过课堂讲授、线上视频学习、真题讲解，学生能够知晓雅思口语部分的基


本形式，能够了解基本解题技巧，能够大体了解雅思口语各部分的常见话题分类。能够适应雅思口


语回答在形式、结构、时间控制等方面的特点。


思政融合点 **5** ：在口语部分的讲解中，既会帮助学生扎实掌握英语语言知识，培养学生英语语


言应用能力和思辨能力，增强学生跨文化交际能力和团队协作能力，又会潜移默化地引导学生积极


践行社会主义核心价值观，厚植家国情怀，鼓励学生用英语讲好中国故事，传播好中国声音，传承


中华文脉。


**6.** 自主练习优化与探讨


（ **1** ）主要内容


    - 雅思考试自主复习的方式与方法


    - 雅思考试自主复习可能存在的误区


（ **2** ）教学重点： 雅思考试自主复习的方式与方法。


（ **3** ）教学难点： 雅思考试自主复习的方式与方法。


（ **4** ）教学要求： 通过智慧树平台、我校泛雅平台布置模拟题，供学生进行自主模拟训练，通


过课堂活动指导学生进行技巧演练，为后期的自主复习打下基础。


三、 实践环节及要求


本课程实践环节包括课内实践和课外实践两个部分。课内实践由教师指导、学生自己组织进行，


要求学生结合课程内容，对雅思考试所需要的的基本语言技能进行模拟式操练，检验和巩固所学的


知识。课外实践主要包括学生独立和以学习小组形式完成的各项课外任务。

八、 课程目标达成度定量评价


372


在课程结束后，需要对每一个课程目标（含思政课程目标）进行达成度的定量评价，用以实现


课程的持续改进。


课程目标达成度的定量评价算法：


1、使用教学活动成绩或期末考试部分题目得分率作为评价项目，对某个课程目标进行达成度


的定量评价；


2、为保证考核的全面性和可靠性，要求对每一个课程目标的评价项目选择至少两种；


3、根据施教情况，评价项目可以由教师自行扩展，权重比例可以由教师自行设计；


4、对某一个课程目标有支撑的各评价项目权重之和为 1；


5、使用所有学生（含不及格）的平均成绩计算。
