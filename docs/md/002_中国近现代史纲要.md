一、课程目标


《中国近现代史纲要》属于高校“思想政治理论课”（简称“思政课”）之一，课程从历史角度承


担着思想政治理论教育的功能，是培养大学生历史素养和政治素养的基础课，对当代大学生开阔历


史视野，辨别历史是非具有非常重要的作用。通过课程教学，讲授中国近代以来抵御外来侵略、争


取民族独立、实现人民解放的历史，努力达到以下课程目标：


课程目标 **1** （知识）： 帮助学生了解国史、国情，深刻领会历史和人民怎样选择了马克思主义，


怎样选择了中国共产党，怎样选择了社会主义道路，怎样选择了改革开放。


课程目标 **2** （能力）： 培养学生运用科学的历史观和方法论分析问题、解决问题，提高学生逻


辑思考和辨别是非的能力。


课程目标 **3** （素养）： 提升学生的人文精神和人文素养，引导学生形成良好的事业心和责任感，


树立正确的世界观、人生观、价值观。

本课程详细教学内容和方法阐述如下：


**1.** 课程导言


（ **1** ）主要内容： 中国近现代史纲要的课程性质、课程要求、课程目标、考核方式等。


（ **2** ）教学重点： 如何理解思想政治理论课的课程性质和课程功能。


（ **3** ）教学难点： ①思政课与历史课的区别与联系；②思政课对大学生有什么用。


（ **4** ）教学要求： 首先，说明中国近现代史纲要课程的“思想政治理论课”性质，回答学生对于课程


功用的疑问。其次，对课程教学内容加以简单总结，要求学生掌握基本的历史脉络。再次，对课程教学


方法、考核方式、参考书等加以说明。


**2.** 进入近代后中华民族的磨难与抗争


（ **1** ）主要内容： 鸦片战争前的中国和世界；近代中国的社会性质、主要矛盾及其基本特征；近代


中国的两大历史任务及其相互关系；资本-帝国主义对中国的军事侵略、政治控制、经济掠夺、文化渗


透的史实；中国人民和清朝爱国官兵反对外来侵略斗争的英勇事迹，中国反对外来侵略失败的原因。


（ **2** ）教学重点： ①近代中国的主要矛盾和社会性质、历史任务；②资本-帝国主义的侵略给中国带


来的影响。


（ **3** ）教学难点： ①近代中国人民反侵略战争及其失败的原因和教训；②近代中国社会衰落的原因。


（ **4** ）教学要求： 通过本章的学习，使学生掌握伴随资本-帝国主义的入侵，中国社会主要矛盾、社


会性质的变化。同时了解当代中国社会主要矛盾的转变。通过本章的学习，使学生正确认识资本-帝国


主义对中国的侵略，廓清“侵略有功论”。了解人民群众和爱国官兵的反抗斗争是中华民族历尽劫难而不


亡的决定性因素，同时了解社会制度腐败、经济技术落后是中国人民反对外国侵略失败的根本原因。


**3.** 不同社会力量对国家出路的早期探索


（ **1** ）主要内容： 介绍农民阶级、地主阶级洋务派及资产阶级维新派对国家出路的早期探索；分析


农民阶级、地主阶级洋务派及资产阶级维新派探索失败的原因及经验教训。


（ **2** ）教学重点： 农民阶级、地主阶级洋务派及资产阶级维新派探索失败的原因。


（ **3** ）教学难点： ①如何理解农民阶级的阶级局限性，不能领导中国走向独立富强；②洋务运动的


起因、洋务派的思想和洋务运动的影响对中国近代化的作用。


（ **4** ）教学要求： 通过本章的学习，使学生在了解基本史实的基础上，明白中国社会是如何艰难地


转变与进步的。


**4.** 辛亥革命与君主专制制度的终结


（ **1** ）主要内容： 介绍以孙中山为首的资产阶级革命派为在中国建立资产阶级民主共和国的奋斗过


程；袁世凯及北洋军阀的专制统治；分析辛亥革命的意义及资产阶级共和国在中国行不通的原因。

（ **2** ）教学重点： ①辛亥革命的意义和失败原因；②袁世凯及北洋军阀专制统治的反动统治。


（ **3** ）教学难点： ①对辛亥革命的评价；②资产阶级共和国方案在中国行不通的原因。


（ **4** ）教学要求： 通过对本章的学习，使学生懂得革命不是任何人主观意志的产物，“告别革命论”


是历史虚无主义思潮的表现，辛亥革命是中国近现代历史上第一次历史性的巨变。同时，也使学生明白


民族资产阶级是一个具有两面性的阶级，动摇和妥协是其阶级的局限性。


**5.** 中国共产党成立和中国革命新局面


（ **1** ）主要内容： 中国先进分子选择马克思主义的历史必然性；中国共产党“应运而生”；“不忘初心，


牢记使命”；第一次国共合作和国民革命运动。


（ **2** ）教学重点： ①中国共产党“应运而生”、建党精神；②第一次国共合作与国民革命的兴起及其


失败的原因。


（ **3** ）教学难点： ①五四运动的必然性及其在中国近代史上的历史作用与意义；②五四运动与中国


共产党及中国革命的关系。


（ **4** ）教学要求： 通过本章的学习，使学生懂得：中国先进分子选择马克思主义，是经过长期的、


艰苦的探索之后的必然结果，中国共产党是马克思主义与中国工人运动相结合的产物，它的出现，是中


国开天辟地的大事变，从此，中国革命的面貌焕然一新。


**6.** 中国革命的新道路


（ **1** ）主要内容： 农村包围城市、武装夺取政权的理论及对革命新道路的艰苦探索；长征精神。


（ **2** ）教学重点： ①中国共产党探索开辟革命新道路的历程；②新民主主义革命理论的形成。


（ **3** ）教学难点： ①宁汉合流后国民党政权及其性质；②中国共产党领导的革命及其性质和意义；


③长征精神及历史意义。


（ **4** ）教学要求： 通过本章的学习，使学生理解进行土地革命战争的必要性、正义性和进步性，懂


得把马克思主义普遍真理同中国革命具体实践相结合的极端重要性。


**7.** 中华民族的抗日战争


（ **1** ）主要内容： 日本灭亡中国的罪恶图谋和野蛮暴行；抗日民族统一战线的建立、巩固及意义；


国民党在抗日战争中的地位和作用；共产党是中国人民抗日战争的中流砥柱；中国人民的抗日战争在世


界反法西斯战争中的地位和作用；抗日战争胜利的意义、原因和基本经验。


（ **2** ）教学重点： ①日本发动侵华战争的原因；②抗日战争的伟大历史意义。


（ **3** ）教学难点： ①抗日战争的三个阶段及其主要作战情况；②中国共产党在抗日战争的中流砥柱


作用。


（ **4** ）教学要求： 通过本章的学习，使学生认识到日本帝国主义对中国的侵略是蓄谋已久的既定国


策，我们要铭记历史，防止悲剧的重演。同时也懂得，抗日战争是在抗日民族统一战线的旗帜下取得胜


利的，兄弟齐心其利断金，体现了伟大的中华民族精神，也是我们今天实现中华民族伟大复兴不可或缺


的。
